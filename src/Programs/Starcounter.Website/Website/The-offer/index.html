Is Starcounter for you?

• Is performance important?
• Does your application handle live data?
• Should the data be updatable by many?
• Need truly consistent data? (not eventually consistent data)

Then you should definitely try Starcounter

The Starcounter database is built with the real-time systems of today in mind. Real-time systems supporting thousands or millions of simultaneous users. Systems like online stores, location-based services or banking applications. Systems where the users need to access same data, and also should be able to update same data. Where numbers need to be correct, independent of if several users are buying from the same stack, or if one changes his mind or another gets his connection cut off half way through.

The numbers need to be correct to ensure consistency. For these systems where performance is of essence or performance might be a challenge today, this is where Starcounter fits in. Starcounter combines high performance with full guarantee that database transactions are processed reliably and the data is kept consistent.

High performance for Starcounter is counted in several millions of transactions per second. This is the order of magnitude of performance needed to create better software applications and powerful Internet services of the future. Read the success stories about Adtoma and Heads.
